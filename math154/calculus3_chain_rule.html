Example - Ski mountain



Consider skiing on the mountain above with \(z\) representing elevation and the black curve representing the path taken. How does elevation change with respect to time?

  1. Consider a new simpler mountain given by \(z = e^{-x^2 -y^2}\) and ski path given by \(x(t) = 1 - e^{-t}, y(t) = -t^4 + t^2\). Find \(\frac{dz}{dt}\).
  2. What is the slope when \(t = 2\)?
  3. What if the path were \(x(t) = \cos t, y(t) = \sin t\)? Find \(\frac{dz}{dt}\). Interpret your solution.

Example - more chain rule

Let \(z\equiv z(x,y), x\equiv x(s,t), y \equiv y(s,t)\). We can't find \(\frac{dz}{ds}\). We need to find \(\frac{\partial z}{\partial s}\). Come up with a formula for \(\frac{\partial z}{\partial s}\).

  1. Example. \(\displaystyle z = e^x \sin y\) with \(x = st^2, y=s^2t\). Find \(\frac{\partial z}{\partial s}\) and \(\frac{\partial z}{\partial t}\).

Example - implicit partial derivatives

Consider \(\sin(xy) + \pi y^2 - x = 0\).

  1. Use implicit differentiation to find \(\frac{dy}{dx}\).
  2. Now let \(F(x,y) = \sin(xy) + \pi y^2 - x = 0\) or simply \(F(x,y) = 0\). Implicitly differentiate with respect to \(x\) and solve for \(\frac{dy}{dx}\).
  3. Try the same thing but with \(G(x,y) = x^2 - 2y^2 - 1 = 0\) implicitly differentiate to find \(\frac{dy}{dx}\).

What if we have \(F(x,y,z) = 0\) instead? What changes?

  1. Assume \(z\equiv z(x,y)\). Implicitly differentiate with respect to \(x\) (need partial derivatives now) and solve for \(\frac{\partial x}{\partial x}\).
  2. Find \(\frac{\partial x}{\partial x}\) if \(z(x,y)\) is defined implicitly by \(xy + xz + yz = 3\).

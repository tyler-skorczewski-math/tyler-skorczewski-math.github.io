Example

Consider the function \(f(x) = x^2\). We know the limit \(\displaystyle \lim_{x \to 2} f(x) = 4\). Consider the \(\epsilon - \delta\) definition of limit and answer the following questions.

  1. If we let \(\displaystyle \epsilon = 1\), what is the largest \(\delta\) could be?

  2. If we let \(\displaystyle \epsilon = 0.1\), what is the largest \(\delta\) could be?

  3. If we let \(\displaystyle \epsilon = 0.01\), what is the largest \(\delta\) could be?

  4. If we let \(\displaystyle \epsilon > 0\) be arbitary, what is the largest \(\delta\) could be?

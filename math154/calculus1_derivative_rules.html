More derivative rules

  1. Sum rule for derivatives: Let \(h(x) = f(x) + g(x)\). Then \(h'(x) = f'(x) + g'(x)\) provided the derivatives exist.

  2. Constant multiple rule: Let \(k\) be a constant number and \(r(x) = k f(x)\). Then \(r'(x) = k f'(x)\) provided \(f'(x)\) exists.

  3. True or False. Let \(h(x) = f(x) - g(x)\). Then \(h'(x) = f'(x) - g'(x)\). Why or why not?

Example - going backwards

Power Rule for Derivatives: If \(f(x) = x^n\), then \(f'(x) = nx^{n-1}\).

Whenever we find a rule for derivatives, we also find a rule for antiderivatives.

Power Rule for Antiderivatives: If \(f(x) = x^n\) and \(n \neq -1\), then the function \(F(x)\) such that \(F'(x) = f(x)\) is \(F(x) = \frac{1}{n+1}x^{n+1} + C\) for any constant \(C\).

Find \(F(x)\) such that \(F'(x) = f(x)\) for the functions below.

  1. \(\displaystyle f_1(x) = 2x\)

  2. \(\displaystyle f_2(x) = x^2\)

  3. \(\displaystyle f_3(x) = x + 2x^2 + 8x^4 + 4\)


  1. \(\displaystyle F_1(x) = x^2 + C\)

  2. \(\displaystyle F_2(x) = (1/3) x^3 + C\)

  3. \(\displaystyle F_3(x) = (1/2)x^2 + (2/3)x^3 + (8/5)x^5 + 4x + C\)

Question: Let \(f'(x) = x^2\) and \(g'(x) = x^2\). Does \(f(x) = g(x)\)?

No. \(f\) does not necessarily equal \(g\). The constants could be different, e.g. \(f(x) = (1/3)x^3 + 1\) and \(g(x) = (1/3)x^3 + 2\).

Example - product rule

Let \(f(x) = x\) and \(g(x) = 2x\). Does \((fg)' = f'g'\)?

Product Rule for derivatives: Let \(u(x)\) and \(v(x)\) be differentiable functions. Then let \(p(x) = u(x)v(x) = (uv)(x)\). Then \(p\) is differentiable and \(\frac{dp}{dx} = p'(x) = u'(x)v(x) + u(x)v'(x)\).

Example - quotient rule

Quotient rule for derivatives: Let \(\displaystyle q(x) = \frac{u(x)}{v(x)}\) where \(u\) and \(v\) are differentiable. Then \(\displaystyle q' = \frac{u' v - uv'}{v^2}\).

Prove this is true by letting \(u = qv\) and using the product rule.

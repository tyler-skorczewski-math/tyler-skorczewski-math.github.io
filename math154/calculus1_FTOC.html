This leads to the following BIG IDEA ...

Theorem - FTOC

Theorem (The Fundamental Theorem of Calculus): Let \(f\) be any continuous function defined on an open interval \(I\) containing point \(a\). Then we have

\(\displaystyle \frac{d}{dx} \int_a^x f(t)~dt = f(x)\)
or
\(\displaystyle A_{f,a}(x) = \int_a^x f(t) dt\) and \(\displaystyle \frac{dA_{f,a}}{dx} = f(x)\).


Remark: this says the integral of a function is an antiderivative!

How does this work with our previous example?

  • \(\displaystyle A_{f,0}(x) = \int_0^x f(t) dt = \int_0^x t^2~dt = \frac{1}{3}x^3 + C\)
  • Note \(C = 0\) because \(A_{f,0}(0) = \int_0^0 f(t) dt = 0 = \frac{1}{3}0^3 + C \Rightarrow C = 0\).

FTOC part two

Another form of the FTOC looks like \(\displaystyle \int_a^b f(x)~dx = F(b) - F(a)\). This is saying that to find the area under the graph of \(f(x)\) from \(a\) to \(b\) we can evaluate the difference in the antiderivative, \(F\), at \(b\) and \(a\). We also use the notation \(\displaystyle \int_a^b f(x)~dx = F(b) - F(a) = F(x)]^b_a\).

Example: Evaluate \(\displaystyle \int_1^4 (5x + x^{3/2})~dx\).

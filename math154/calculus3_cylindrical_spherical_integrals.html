Voting questions -

Which of the following is equivalent to the integral \(\displaystyle \int_{-5}^5 \int_0^3 \int_{-\sqrt{25-x^2}}^{\sqrt{25-x^2}} x~dy~dz~dx\)?

  1. \(\displaystyle \int_0^3 \int_0^3 \int_0^{\pi} r^2 \cos \theta~d\theta~dz~dr\)
  2. \(\displaystyle \int_0^5 \int_0^3 \int_0^{\pi} r^2 \cos \theta~d\theta~dz~dr\)
  3. \(\displaystyle \int_0^3 \int_0^5 \int_0^{2\pi} r \cos \theta~d\theta~dz~dr\)
  4. \(\displaystyle \int_0^5 \int_0^3 \int_0^{2\pi} r^2 \cos \theta~d\theta~dz~dr\)
  5. none of the above

Which of the following gives the volume of the unit sphere?

  1. \(\displaystyle \int_0^{2\pi} \int_0^{2\pi} \int_0^{1} d\rho~d\theta~d\phi\)
  2. \(\displaystyle \int_0^{\pi} \int_0^{2\pi} \int_0^{1} d\rho~d\theta~d\phi\)
  3. \(\displaystyle \int_0^{\pi} \int_0^{2\pi} \int_0^{1} \rho^2 \sin\phi d\rho~d\theta~d\phi\)
  4. \(\displaystyle \int_0^{\pi} \int_0^{2\pi} \int_0^{1} \rho^2 \sin\phi d\rho~d\phi~d\theta\)
  5. \(\displaystyle \int_0^{\pi} \int_0^{2\pi} \int_0^{1} \rho d\rho~d\theta~d\phi\)
  6. none of the above

Example - One volume many ways

Consider finding the volume between the surfaces \(z = x^2 + y^2\) and \(z = \sqrt{x^2 + y^2}\).

  1. Sketch the region.
  2. Use the calc 2 method of disks/washers to set up an integral that represents this volume.
  3. Set up a triple iterated integral in rectangular coordinates that represents this volume.
  4. Set up an integral in polar coordinates that represents this volume.
  5. Set up an integral in cylindrical coordinates that represents this volume.
  6. Find the volume by evaluating one of your integrals above.
  7. What does each approach have in common? What is different? Is one way better/worse? Why?

Example - Spherical integrals

  1. Set up an integral that represents the average distance to the origin for points in the 'ice cream cone' region bounded by the hemisphere \(z=\sqrt{8 - x^2 - y^2}\) and the cone \(z = \sqrt{x^2 + y^2}\).

  2. A spherical shell centered at the origin has an inner radius of 6cm and and outer radius of 7cm. The density varies linearly with distance from the center. At the inner surface \(\rho = 9\)gm/cm\(^3\). At the outer surface \(\rho = 11\)gm/cm\(^3\). Set up an integral for the mass of this solid.

  3. Show \(\displaystyle \iiint_T x~dV =\frac{\pi}{16}\) where \(T\) is the region in the first octant bounded by \(x^2 + y^2 + z^2 = 1\).

Example - More integrals

Which integral below evaluates easily? Why? Confirm your answer by performing the calculation and do the calculation for the other integral.

  1. Evaluate \(\displaystyle \iiint_T \sqrt{x^2 + y^2}~dV\), where \(T\) is the region bounded by \(x^2 + y^2 = 1\) and \(z=1\) and \(z=3\).

  2. Evaluate \(\displaystyle \iiint_T y~dV\), where \(T\) is the region bounded by the \(xy\) plane and \(z = \sqrt{1- x^2 - y^2}\).

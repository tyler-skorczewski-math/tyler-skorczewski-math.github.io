Example - Logarithmic differentiation

Let \(\displaystyle f(x) = x^2e^{-x^3}\sin(2\pi x)\).

  1. Find \(\frac{df}{dx}\) using the product rule.

  2. Now go back and write down \(\displaystyle f(x) = x^2e^{-x^3}\sin(2\pi x)\).

  3. Take the \(\ln\) of both sides of this equation.

  4. Apply properties of logarithms.

  5. Use implicit differentiation to take derivative with respect to \(x\).

  6. Solve for \(\frac{df}{dx}\). This process is known as logarithmic differentiation.

Example -

Use logarithmic differentiation to find the derivatives of the functions below.

  1. \(\displaystyle f(x) = \frac{x^2 - 5x + 1}{(x^4 - 3x)(3x^3 - 2x^2)}\)

  2. \(\displaystyle g(x) = \sin^4 (x^3)(x^3 - x + 1)\)

  3. \(\displaystyle h(x) = \frac{e^{5x}}{\cos^2 (2x + 1)}\)

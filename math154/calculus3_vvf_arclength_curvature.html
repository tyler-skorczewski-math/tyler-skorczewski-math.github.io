Theorem

Consider a curve given by \(\displaystyle \vec{r}(t) = \begin{bmatrix}f(t)\\g(t)\\h(t)\end{bmatrix}\) on interval, \(I\), given by \(a \leq t \leq b\). Draw a picture and describe geometrically and physically why the arc length of this curve is given by the following terms.

  1. \(\displaystyle L = \int_I dL\)
  2. \(\displaystyle L = \int_a^b |\vec{r}'|~dt\)
  3. \(\displaystyle L = \int_a^b \sqrt{f'^2 + g'^2 + h'^2}~dt\)
  4. \(\displaystyle L = \int_I \sqrt{dx^2 + dy^2 + dz^2}\)
  5. \(\displaystyle L = \int_a^b \sqrt{\left(\frac{dx}{dt}\right)^2 + \left(\frac{dy}{dt}\right)^2 + \left(\frac{dz}{dt}\right)^2}~dt\)

Curvature


Consider the picture above showing a curve with several unit tangent vectors sketched. The curvature of this path is given by \(\displaystyle \kappa = \left| \frac{dT}{ds}\right|\) where \(s\) is the arc length variable.

  1. What is the formula above saying about curvature? (And no- you can't just claim it is describing how much the path is curved.)
  2. Rewrite this formula to only include derivatives with respect to a time-like variable and not arc length.
  3. What are the units of curvature? (hint - see answer above)
  4. Use the fact that \(\displaystyle \vec{T} = \frac{\vec{r}'}{|\vec{r}'|}, |\vec{r}'| = \frac{ds}{dt}\) to find a formula for curvature that only relies on derivatives of \(\vec{r}\) with respect to time.
-
  • We have that curvature is \(\displaystyle \kappa = \left| \frac{d\vec{T}}{ds} \right| \).

  • We want a formula that uses derivatives with respect to \(t\).
  • \( \displaystyle \kappa = \left| \frac{d\vec{T}}{ds} \right| = \left| \frac{\frac{d\vec{T}}{dt}}{\frac{ds}{dt}} \right| = \frac{| \vec{T}' |}{|\vec{r}'|} \)

  • We know that \(\displaystyle \vec{T} = \frac{\vec{r}'}{|\vec{r}'|}, |\vec{r}'| = \frac{ds}{dt}\) so we have the following.
  • \(\displaystyle \vec{r}' = |\vec{r}'|\vec{T} = \frac{ds}{dt}\vec{T}\)

    \(\displaystyle \vec{r}'' = \frac{d^2s}{dt^2}\vec{T} + \frac{ds}{dt}\vec{T}'\)

    \(\displaystyle \vec{r}' \times \vec{r}'' = (\vec{r}' \times \frac{d^2s}{dt^2}\vec{T}) + (\vec{r}' \times \frac{ds}{dt}\vec{T}') \)

  • Note that the first term is zero (why?) and \(\displaystyle \vec{r}' = \frac{ds}{dt}\vec{T}\) so

    \(\displaystyle \vec{r}' \times \vec{r}'' = \left(\frac{ds}{dt}\right)^2 \left( \vec{T} \times \vec{T}' \right) \).

  • Now since \( |\vec{T}| = 1 \) (constant) we know \(\vec{T} \perp \vec{T}'\) so \( |\vec{T} \times \vec{T}'| = |\vec{T}| |\vec{T}'|\).

  • Putting everything together we can get our result:

    \(\displaystyle |\vec{r}' \times \vec{r}'' | = \left(\frac{ds}{dt}\right)^2 |\vec{T}'| \)

    \(\displaystyle |\vec{T}'| = \frac{|\vec{r}' \times \vec{r}'' |}{|\vec{r}'|^2}\)


    \(\displaystyle \kappa = \frac{| \vec{T}' |}{|\vec{r}'|} = \frac{|\vec{r}' \times \vec{r}'' |}{|\vec{r}'|^3}\)

Example - planar curvature formulas

Consider a curve in \(\mathbb{R}^2\) given parametrically by \(x=f(t), y=g(t)\).

  1. Find a formula for curvature in terms of derivatives of \(f\) and \(g\).
    1. Use this formula to find the curvature of the path described by an ellipse with radius 3 in the \(x\) direction and 2 in the \(y\) direction.
      -
      • \(\displaystyle \vec{r} = \begin{bmatrix}f\\g\end{bmatrix} \Rightarrow f(t) = 3\cos(t), g(t) = 2\sin(t) \)

      • \(\displaystyle \kappa = \frac{|f'g'' - f''g'|}{ \left[ f'^2 + g'^2 \right] ^{3/2}} \)

      • \(\displaystyle f' = -3\sin t, f'' = -3 \cos t\)

      • \(\displaystyle g' = 2 \cos t, g'' = -2 \sin t\)

      • \(\displaystyle \kappa = \frac{|6\sin t \cos t + 6 \sin t \cos t|}{\left[ 9 \sin^2 t + 4 \cos^2 t \right]^{3/2}} = \frac{|6\sin (2t) |}{\left[ 9 \sin^2 t + 4 \cos^2 t \right]^{3/2}} \)

  2. Can you find a formula for the curvature of the graph of a function \(y = f(x)\)?
    1. What is the curvature of the graph of \(y = x^2\)?
      -
      • \(\displaystyle \kappa = \frac{|f''(x)|}{ \left[ 1 + f'(x)^2 \right] ^{3/2}} \)

      • \(\displaystyle f'(x) = 2x, f''(x) = 2\)

      • \(\displaystyle \kappa = \frac{2}{\left[ 1 + 4x^2 \right]^{3/2}} \)

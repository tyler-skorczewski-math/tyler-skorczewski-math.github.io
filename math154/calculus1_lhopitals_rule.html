Calc I l'Hopital's rule

Example and theorem

Consider the function \(f(x) = \ln ( \cos (x)) \). What happens to the values of \(f(x)\) as \(x \to 0\)?

Answer
\(\displaystyle \lim_{x \to 0} \ln (\cos x) = \ln ( \lim_{x \to 0} \cos x) = \ln( \cos 0) = \ln (1) = 0\)


Theorem Let \(f\) and \(g\) be functions such that \(\lim_{x\to a} g(x) = b\) and \(f\) is continuous at \(b\). Then \(\lim_{x\to a}f(g(x)) = f(\lim_{x\to a}g(x)) = f(b)\). Note we saw this theorem earlier, but now we are allowing \(a\) to be \(\pm \infty\) too.

Example: Evaluate \(\displaystyle \lim_{x \to \infty}e^{\frac{x^3 - \sqrt{x}}{-x - x^3}}\).

Answer
\(\displaystyle \lim_{x \to \infty}e^{\frac{x^3 - \sqrt{x}}{-x - x^3}} = e^{\lim_{x\to \infty} \frac{x^3 - \sqrt{x}}{-x - x^3}} = e^{-1} = \frac{1}{e}\)


Example

What is \(\displaystyle \lim_{x \to \infty}x e^{-x}\)?

Note as \(x \to \infty\), we have \(x \to \infty\) but also \(e^{-x} \to 0\), so we have something that looks like \(0 \cdot \infty\). Anything multiplied by 0 is supposed to be zero, but anything 'multiplied by infinity' is supposed to be infinite. What is happening here?

\(0 \cdot \infty\) is an example of an indeterminate form. We handle limits that are indeterminate forms using l'Hopital's rule.

Theorem (l'Hopital's rule) Let \(f\) and \(g\) be differentiable functions such that as \(x \to a\) (or \(\pm \infty\) ) either

  1. \(\displaystyle \lim_{x\to a} \frac{f(x)}{g(x)} \to \frac{\infty}{\infty}\) or,
  2. \(\displaystyle \lim_{x\to a} \frac{f(x)}{g(x)} \to \frac{0}{0}\).

Then \(\displaystyle \lim_{x\to a} \frac{f(x)}{g(x)} = \lim_{x\to a} \frac{f'(x)}{g'(x)}\).

This doesn't appear to help us since \(xe^{-x} \to 0 \cdot \infty\) not one of the forms from the theorem above. What can we do?

Example - Line integral

Definition Line integral \( \displaystyle \int_C \vec{F}\cdot\vec{dr} = \lim_{||\Delta x \to 0} \sum_{i=1}^{n} \vec{F}(\vec{r}_i)\cdot \Delta \vec{r}_i \)

  1. Example: Find the line integral of the constant vector field \( \vec{F} = 1\hat{x} + 2\hat{y}\) along the path from (1,1) to (10,10) by going from (1,1) to (10,1) then to (10,10).
  2. Example: What if we want to go from (1,1) to (2,4) along the parabola \(y=x^2\) through this vector field?
  3. Remark: line integrals are independent of parameterization.

Example - FTOC Line Integrals

Recall the Fundamental Theorem of Calculus from Calculus 1: \(\displaystyle \int_a^b f'(t) dt = f(b) - f(a)\).

Theorem (Fundamental Theorem of Line Integrals) \(\displaystyle \int_C \nabla f \cdot \vec{dr} = f(Q) - f(P)\) where \(P\) and \(Q\) are the starting and ending points of a smooth oriented curve \(C\).

Question: What does this mean if the path is a closed curve?

  1. Consider \(\displaystyle \vec{F} = \begin{bmatrix}1\\2\end{bmatrix}\) and the path from (1,1) to (2,4) along \(y=x^2\). What is \(\displaystyle \int_C \vec{F}\cdot d\vec{r}\)?
  2. Let \(f(x,y) = (1/2)x^2 + (1/2)y^2\) and \(\vec{F} = \nabla f\). Let \(C\) be the circular arc from (0,1) to (1,0). What is \(\displaystyle \int_C \vec{F}\cdot d\vec{r}\)?

Path Independence

Definition A vector field \(\vec{F}\) is said to be path independent if for any two points \(P\) and \(Q\), the line integral \(\displaystyle \int_C \vec{F}\cdot~d\vec{r} \) has the same value for any piecewise smooth oriented path \(C\) from \(P\) to \(Q\) in the domain of \(\vec{F}\).

Remark All gradient fields \(\vec{F} = \nabla f\) are path independent. Why?

Question How does this idea connect to gravity/gravitational fields?

Theorem A continuous vector field \(\vec{F} \) defined on an open region is path independent if and only if \(\vec{F}\) is a gradient field.

Example Show \(\displaystyle \vec{F} = \begin{bmatrix}y \cos (x) \\ \sin (x) + y\end{bmatrix}\) is path independent by finding a potential function.

Example - Flow lines

Definition A flow line of a vector field \(\vec{F}: X \subset \mathbb{R}^n \to \mathbb{R}^n\) is a differentiable path \(\vec{r}: I \to \mathbb{R}^n\) such that \(\vec{r}'(t) = \vec{F}(\vec{r}(t))\).

  1. Find the flow lines of \(\displaystyle \vec{F}(x,y) = y \hat{x} -x \hat{y}\).
  2. Which curve is not a flow line of the vector field \( \displaystyle \vec{F}(x,y) = x \hat{x} + y \hat{y}\)?
    1. \(\displaystyle \vec{r}(t) = e^t \hat{x} + e^t \hat{y}\)

    2. \(\displaystyle \vec{r}(t) = e^t \hat{x} + 2e^t \hat{y}\)

    3. \(\displaystyle \vec{r}(t) = 3e^t \hat{x} + 3e^t \hat{y}\)

    4. \(\displaystyle \vec{r}(t) = 2e^t \hat{x} + e^{2t} \hat{y}\)

Example -Gradient fields

Definition If a vector field is of the form \(\displaystyle \vec{F} = \nabla f \) for some scalar function \(f\), we say \(\vec{F}\) is a conservative field (or gradient field), and \(f\) is a potential function for \(\vec{F}\).

Note In physics it is common to have \(\displaystyle \vec{F} = -\nabla f \).

  1. Let \(f(x,y) = \cos(3x^2) + ye^y\). Find \(\vec{F} = \nabla f\).
  2. What is the potential function for the vector field \(\vec{F} = 2y\hat{x} + 2x\hat{y}\)?
    1. \( f(x, y) = 4xy\)
    2. \( f(x, y) = 2x^2 + 2y^2\)
    3. \( f(x, y) = 2xy\)
    4. This is not a conservative vector field.
  3. Is \(\vec{F} = x^2y \hat{x} - 2xy \hat{y}\) a conservative field?

Example - Theorem

Theorem Let \(\vec{F}\) be a vector field in \(\mathbb{R}^2\) such that the component functions of \(\vec{F}\) have continuous second order mixed partial derivatives on the domain of \(\vec{F}\). If \(\displaystyle \vec{F} = \begin{bmatrix} P(x,y) \\ Q(x,y) \end{bmatrix}\) is a conservative field then \(P_y = Q_x\).

  • Show this theorem holds for \(\vec{F} = 2y\hat{x} + 2x\hat{y}\).
  • What are the converse, inverse, and contrapositive of this statement?

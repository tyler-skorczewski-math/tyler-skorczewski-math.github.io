Tangent Planes

Warmup

  1. True or False. There exists a function \(f(x, y)\) with \(f_x = 2y\) and \(f_y = 2x\).

  2. True or False. If \( \frac{\partial f}{\partial x} = \frac{\partial f}{\partial y}\) everywhere, then \(f(x, y)\) is constant.

  3. True or False. \(f = x^{0.4}y^{0.6}\) solves Euler's equation \(xf_x + yf_y = f\).

  4. True or False. \(f = x+y+1\) solves Euler's equation \(xf_x + yf_y = f\).

Tangent Planes

$$z - z_0 = f_x(x_0,y_0) (x-x_0) + f_y(x_0,y_0) (y-y_0)$$

Examples

Find tangent planes for the functions below at the specified points.
  1. \( \displaystyle z = g(x,y) = \sqrt{4-x^2 - y^2}\) at point \((1,1,\sqrt{2})\)
  2. \( \displaystyle z = f(x,y) = 32 - 3x^2 - 4y^2\) at point (2,1,16)

Linear Approximation in 2D

Let \(\displaystyle f(x,y) = \frac{5}{x^2 + y^2}\).

  1. What is the tangent plane at (-1,2)?
  2. Use a linear approximation to estimate \(f(-1.05,2.1)\) using \(f(-1,2)\).

Differential Definition

Let \(z = f(x,y)\) be a function of two variables with \( (x_0,y_0) \) in the domain of \(f\). Then the total differential of \(z\) is given $$dz = f_x(x_0,y_0) dx + f_y(x_0,y_0)dy.$$

Example - Resistors


Consider 3 resistors in parallel. The total resistance is given by $$\frac{1}{R_{tot}} = \frac{1}{R_{1}} + \frac{1}{R_{2}} + \frac{1}{R_{3}}.$$ Let \(R_1 = 20 \Omega \pm 1 \Omega\), \(R_2 = 10 \Omega \pm 0.5 \Omega\), and \(R_1 = 2 \Omega \pm 0.2 \Omega\).

  1. Calculate \(R_{tot}\).
  2. Calculate the total differential \( dR_{tot}\)
  3. Use your answer above to estimate the error in \(R_{tot}\).

Warmup/Formulas

Derivative Rule Antiderivative Rule
\(\displaystyle \frac{d}{dx} e^x = e^x\) \(\displaystyle \int e^x~dx = e^x +C\)
\(\displaystyle \frac{d}{dx} a^x = a^x \ln(a)\)
 \(\displaystyle \int a^x~dx = \frac{1}{\ln(a)} a^x +C\)
\(\displaystyle \frac{d}{dx} \ln(x) = \frac{1}{x}\)
 \(\displaystyle \int \frac{1}{x}~dx = \ln|x| +C\)
\(\displaystyle \frac{d}{dx} \log_b (x)= \frac{1}{x} \frac{1}{\ln (b)}\)

Make sure you can explain the absolute values in the \(\ln|x| +C\) antiderivative rule.

Examples - Definite Integrals

U-substitution works for definite integrals too. We can deal with the limits of the definite integral two ways.

  1. Compute the associated indefinite integral using a u-sub, expressing the antiderivative in terms of the original variable, then doing the evaluations at the limits.
  2. Do a u-sub to transform the definite integral, but also change the limits when doing the u-sub into the u-variable. Then proceed with the new definite integral.

We can see how this works in the example below.

Calc I Limits Exam Review

  1. You know the following statement is true:

    "If f(x) is a polynomial, then f(x) is continuous."

    Which of the following is also true?
    1. If f(x) is not continuous, then it is not a polynomial.
    2. If f(x) is continuous, then it is a polynomial.
    3. If f(x) is not a polynomial, then it is not continuous.


  2. Find the following limits:
    1. \(\displaystyle \lim_{x \to 2} \frac{3x-18}{2x-12}\).

    2. \(\displaystyle \lim_{\theta \to \pi} \frac{\sin \theta}{\tan \theta}\).

    3. \(\displaystyle \lim_{h \to 0} \frac{\frac{1}{a+h} - \frac{1}{a}}{h}\).

    4. \(\displaystyle \lim_{x \to 3} \frac{x^2 - 4x +3}{x^2 + x -12} \).

    5. \(\displaystyle \lim_{x \to 1} \frac{1}{1-x} - \frac{2}{1-x^2} \).

  3. True or False. \(\displaystyle \lim_{x \to 0} x^2 \cos( \frac{1}{x} )\) exists.
  4. True or False. \(\displaystyle \lim_{\theta \to \pi/2} \cos \theta \cos(\tan\theta)\) exists.
  5. True or False. The function \(f(x) = \cos x - x\) has a root between 0 and \(\pi/2\).
  6. Find the value of \(c\) that makes the function below continuous.
    $$f(x) = \begin{cases}x^2 - c, & x < 5\\ 4x + 2c, & x \geq 5 \end{cases}$$
  7. Suppose it is known for a given \(\epsilon,\delta >0\), if \(|x-3|<\delta\) then \(|f(x) -2| < \epsilon\). Which of the following must also be true?
    1. If \(|x-3|< 2 \delta\) then \(|f(x) -2| < \epsilon\).
    2. If \(|x-3|< \delta\) then \(|f(x) -2| < 2\epsilon\).
    3. If \(|x-3|< \delta/2\) then \(|f(x) -2| < \epsilon/2\).
    4. If \(|x-3|< \delta/2\) then \(|f(x) -2| < \epsilon\).
  8. Use the definition of derivative to find \(f'(x)\) if \(f(x) = 2x^2 + 10x\).

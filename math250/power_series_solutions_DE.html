Power Series Review

Review of some series convergence tests from calculus.



See this page for more practice with power series ideas from calculus.

Definition: If the Taylor series of the function \(f\) converges to \(f(x)\) for all \(x\) in some open interval containing \(a\), then we say that the function \(f\) is analytic at \(x=a\).

Theorem: If \(\displaystyle \sum_{n=0}^{\infty} a_n x^n = \sum_{n=0}^{\infty} b_n x^n\) for every point \(x\) in some open interval \(I\),then \(a_n=b_n\) for all \(n\geq 0\). Moreover, if \(\displaystyle \sum_{n=0}^{\infty} a_n x^n = 0 \) for all \(x\) then \(a_n = 0\) for all \(n\).

Result: We can differentiate power series term by term \(\displaystyle \begin{eqnarray} \frac{d}{dx}\left(\sum_{n=0}^\infty c_n \,x^n\right)&=&\frac{d}{dx}(c_0+c_1x+c_2x^2+c_3x^3+c_4x^4+c_5x^5+\cdots)\\ &=&0+c_1+2c_2x+3c_3x^2+4c_4x^3+5c_5x^4+\cdots\\ &=& \sum_{n=1}^\infty n\,c_n \,x^{n-1}. \end{eqnarray} \)

Note that we have begun our sum at \(n=1\) since the first term (constant term) is zero when we differentiate. However, we don't have to do that, and we can say that \(\displaystyle \frac{d}{dx}\left(\sum_{n=0}^\infty c_n \,x^n\right)=\sum_{n=1}^\infty n\,c_n \,x^{n-1} = \sum_{n=0}^\infty n\,c_n \,x^{n} \) since the \(n=0\) term will be zero anyway on the RHS here.

Shifting indices

Index shifting is one of the 'standard tricks' of mathematics, along with multiplying by one, adding zero, and expanding/regrouping terms in parentheses. The idea is to start a sum at a different value of the index than originally written, but shift the index values so that terms refer to the same quantities. This is very useful to get an equation with many series terms all summing over the same index. If you shift the start backward then you shift the index forwards the same amount in the series.

Example 1: \(\displaystyle \sum_{n=2}^{\infty} c_n x^n = \sum_{n=0}^{\infty} c_{n+2} x^{n+2}\) (shift start back by 2, index forward by 2)

Example 2: \(\displaystyle \sum_{n=3}^{\infty} n(n-2) c_n x^n = \sum_{n=0}^{\infty} (n+3)(n+1)c_{n+3} x^{n+3}\) (shift start back by 3, index forward by 3)

Using power series to solve differential equations

We restrict ourselves to solutions at ordinary points. If we have a DE \(y'' + P(x)y' + Q(x)y = 0\), ordinary points are \(x\) values where \(P(x)\) and \(Q(x)\) are analytic.

The idea to using power series is to plug in \(\displaystyle y = \sum_{n=0}^{\infty} c_n x^n\) and its derivatives into a DE and use properties of power series to find a power series representation of the solution.

Example 1:\((x-2)y' + y = 0\)

Example 2:\((2-x^2)y'' - xy' - 4y = 0\)

Example :\(y'' = y\)

Warmups

  1. Consider the matrix \(A = \begin{bmatrix}-2 & 1 & 1 \\1 & -2 & 1\\ 1 & 1 & -2 \end{bmatrix}\). We know that \(|A - \lambda I| = \lambda(\lambda+3)^2\). Find the eigenvalues and eigenvectors of \(A\).
  2. Show that \(e^{5t}\begin{bmatrix}2\\1\end{bmatrix}\) is a solution to \(\vec{x}' = \begin{bmatrix}4 & 2\\3 & -1 \end{bmatrix} \vec{x}\).

Example 1 -

Consider the DE below. $$y'' - y' - 2y = 0$$

  1. Find the general solution.
  2. Rewrite this second order equation as a system of first order equations.
  3. Can you write the system in matrix-vector form?
  4. Label this system.
  5. What are the eigenvalues and eigenvectors of the matrix? What do you notice?

Example 2 -

  1. Write \(x''' + 7x'' - 2x' = \cos(t)\) as a system of first order equations.
  2. Write \(x' = -2y, y' = 3x\) as a second order equation in one variable.

Example 3 - Diagonalization


Take notes on how diagnonalization can be used to solve the system below. $$x_1' = x_1 + x_2$$ $$x_2' = 4x_1 + x_2$$

  1. Write the system in matrix vector form.
  2. Find the eigenvalues and eigenvectors.
  3. Take notes on how to decouple this problem using a diagonal matrix.

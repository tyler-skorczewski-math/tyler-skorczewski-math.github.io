Linearity

Claim: The Laplace transform is a linear operator.

Example: \(\displaystyle \mathcal{L} (3 + 2t^2) = 3\mathcal{L}(1) + 2 \mathcal{L} (t^2) = 3 \frac{1}{s} + 2 \frac{2}{s^3} = \frac{3}{s} + \frac{4}{s^3}, s>0\).

Going backwards - inverse Laplace transforms

Theorem: Suppose \(f(t)\) and \(g(t)\) are such that \(F(s) = \mathcal{L}(f(t))\) and \(G(s) = \mathcal{L}(g(t))\) exist. If \(F(s) = G(s)\) for \(s > c\) for some \(c\), then \(f(t) = g(t)\) whenever \(f\) and \(g\) are continuous on \([0,\infty)\).

What does this mean? - The Laplace transform is a 1-1 function and an inverse exists, \(\displaystyle \mathcal{L}^{-1}\).

If \(\displaystyle \mathcal{L} (f(t)) = F(s)\), then \(\displaystyle \mathcal{L}^{-1} (F(s)) = f(t)\).

Examples:

  • \(\displaystyle \mathcal{L}^{-1} (\frac{1}{s}) = 1\)

  • \(\displaystyle \mathcal{L}^{-1} (\frac{1}{s^3}) = \mathcal{L}^{-1} ( (\frac{2}{2})\frac{1}{s^3}) = \mathcal{L}^{-1} ((\frac{1}{2})\frac{2}{s^3}) =\frac{1}{2} \mathcal{L}^{-1} (\frac{2}{s^3}) = \frac{1}{2}t^2\)

Remark: the inverse Laplace transform is also a linear operator.


Goal: Using Laplace transforms to solve initial value problems (IVPs)

Before we can work on DEs and IVPs, we need to see what the Laplace transform does to derivatives.




Theorem In general if \(f,f',f'',...,f^{(n-1)}\) are continuous, piecewise smooth, and of exponential order, then \(\mathcal{L} ( f^{(n)}(t))\) exists.

\(\displaystyle \mathcal{L} (f^{(n)}(t)) = \mathcal{L} \left( \frac{d^n f}{dt^n} \right) = s^n F(s) + s^{n-1}f(0) + s^{n-2} f'(0) + ... + f^{(n-1)}(0)\)

A Couple More Theorems

We can look up inverse Laplace transforms in a table, but sometimes we need a form for \(s-a\) instead of \(s\).

Theorem: If \(F(s) = \mathcal{L}(f(t))\) exists for \(s>c\) then \(\mathcal{L}(e^{at}f(t))\) exists for \(s > c + a\) and \(\mathcal{L}(e^{at}f(t)) = F(s-a)\), and \(\mathcal{L}^{-1}(F(s-a)) = e^{at} f(t)\).

PF We know \(\displaystyle F(s) = \int_0^{\infty} f(t) e^{-st}~dt\).
Then \(\displaystyle F(s-a) = \int_0^{\infty} f(t) e^{-(s-a)t}~dt = \int_0^{\infty} f(t) e^{at} e^{-st}~dt = \mathcal{L}(e^{at}f(t)) \).

Note this can be used in both directions, to find Laplace transforms and inverse Laplace transforms.

Example 1: \(\displaystyle \mathcal{L}(t^n) = \frac{n!}{s^{n+1}} \Rightarrow \mathcal{L}(e^{at}t^n) = \frac{n!}{(s-a)^{n+1}} \)

Example 2: \(\displaystyle \mathcal{L}^{-1}(\frac{a}{s^2 +a^2}) = \sin (at) \Rightarrow \mathcal{L}^{-1}(\frac{a}{(s-a)^2 +a^2}) = e^{at}\sin (at) \)

Theorem: Let \(f\) be a piecewise continuous function of exponential order as \(t \to \infty\). Then \(\displaystyle \mathcal{L}(-t f(t)) = F'(s) = \frac{d}{ds}F(s)\). We also have that \(\displaystyle f(t) = \mathcal{L}^{-1}(F(s)) = \frac{-1}{t}\mathcal{L}^{-1}(F'(s))\).
In general, \(\displaystyle \mathcal{L}(t^n f(t)) = (-1)^n \frac{d^n}{ds^n}F(s) = (-1)^nF^{(n)}(s)\).

Example 1: \(\displaystyle \mathcal{L}(t e^{-2t}) = -\frac{d}{ds}\mathcal{L}(e^{-2t}) = -\frac{d}{ds} \frac{1}{s+2} = - \frac{-1}{(s+2)^2} = \frac{1}{(s+2)^2} \)

Example 2: \(\displaystyle \mathcal{L}(t \sin(3t)) = -\frac{d}{ds}\mathcal{L}(\sin(3t)) = -\frac{d}{ds} \left( \frac{3}{s^2 + 3^2}\right) = (-3)(-1) \left( \frac{1}{s^2 + 3^2}\right)^2 (2s) = \frac{6s}{(s^2 + 3^2)^2} \)

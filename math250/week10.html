Example 5 - Practical resonance

Consider a mass-spring system with damping and periodic forcing given by $$mx'' + bx' + kx = F_0 \cos (\omega_f t), \hspace{1cm} m,b,k,F_0 > 0.$$ In general \(x_h\), the solution to the associated homogeneous equation, will always go to zero as \(t \to \infty\). This is because of the damping or friction term, \(b >0\). Our next to find a solution of this DE would be to use the method of undertermined coefficients to find a particular solution, \(x_p\). $$x_0 = A \cos(\omega_f t) + B\sin(\omega_f t)$$ After calculating \(x_p'\) and \(x_p''\), plugging into the DE we solve for \(A\) and \(B\). $$ A = \frac{(k - m\omega_f^2)F_0}{(k-m\omega_f^2)^2 + (b \omega_f)^2} \hspace{1cm} B = \frac{b \omega_f F_0}{(k-m\omega_f^2)^2 + (b \omega_f)^2}$$ $$x_p = A \cos(\omega_f t) + B \sin(\omega_f t)$$ Then we can let \(A = C \cos(\phi)\) and \(B = C \sin(\phi)\) so $$x_p = C\cos(\phi)\cos(\omega_f t) + C\sin(\phi)\sin(\omega_f t).$$ Recall \(\cos(a \pm b) = \cos(a)\cos(b) \mp \sin(a)\sin(b)\) so $$x_p = C\cos(\omega_f t - \phi),$$ where $$C^2 = A^2 + B^2, \hspace{1cm} C = \frac{F_0}{\sqrt{(k-m\omega_f^2)^2 + (b \omega_f)^2}}$$ and $$\tan(\phi) = \frac{B}{A} = \frac{b\omega_f}{k - m\omega_f^2}.$$

Let's consider the amplitude response to be a function of the forcing frequency, \(C \equiv C(\omega_f)\). $$C(\omega_f) = \frac{F_0}{\sqrt{(k-m\omega_f^2)^2 + (b \omega_f)^2}}$$ Plots showing several different values of the other parameters are shown below.

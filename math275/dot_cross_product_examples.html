Dot and Cross Products

Warmup

Assume \(\vec{a} \neq \vec{0}\). Which of the following, if any, are true statements? If it is true, explain why. If not, can you come up with a counterexample that shows why not?

  1. If \(\vec{a} \cdot \vec{b} = \vec{a} \cdot \vec{c}\) then \(\vec{b} = \vec{c}\).
  2. If \(\vec{a} \times \vec{b} = \vec{a} \times \vec{c}\) then \(\vec{b} = \vec{c}\).
  3. If \(\vec{a} \cdot \vec{b} = \vec{a} \cdot \vec{c}\) and \(\vec{a} \times \vec{b} = \vec{a} \times \vec{c}\) then \(\vec{b} = \vec{c}\).

Example 1 - Areas of triangles


A somewhat common problem in computer graphics is to calculate the area of a triangle in 3D space using only its coordinates. Above is an image of the famous Stanford bunny. This is a standard test case for many discrete geometry and computer graphics algorithms.

Example 2 - PiP

Can you determine if the point \(p = (1,1,0)\) is in the triangle with corners \((2,2,0), (-1,0,0), (3,0,0)\)?

  1. First find vectors \(v_1, v_2, v_3\) that point from \(p\) to each corner.
  2. Use the dot product to find the angles between each pair of vectors.
  3. Calculate the winding number and determine if the point is inside the triangle.
  4. If time, try using the cross product to calculate the winding number.
    1. Calculate the cross products \(v_1 \times v_2\), \(v_2 \times v_3\), \(v_3 \times v_1\).
    2. Find the angles between each pair of vectors.
    3. Calculate the winding numer. Is it the same as above? Why or why not?

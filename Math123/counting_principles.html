Counting Principles

Ideas

  • Principle of Multiplication
  • Principle of Addition

Formulas

  • Factorial \( n! = n(n-1)(n-2)...(2)(1)\) and \(0! = 1\).
  • Permutation
    • \(P(n,k)\) = number of ways to order \(k\) things from a set of \(n\)
    • Order is important.
    • \(\displaystyle P(n,k) = \frac{n!}{(n-k)!}\)
  • Combination
    • \(C(n,k)\) = number of ways to choose \(k\) things from a set of \(n\)
    • Order is not important.
    • \(\displaystyle C(n,k) = \begin{pmatrix}n\\k\end{pmatrix} = \frac{n!}{k!(n-k)!}\)
